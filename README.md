# van-riper/claude-plugins

A personal Claude Code plugin marketplace.

## Install

```text
/plugin marketplace add van-riper/claude-plugins
/plugin install pythonicator@van-riper
```

## Plugins

- **pythonicator** — enforces the house Python styleguide on agentic
  Python work: a derived canon, a ruff and ty PostToolUse hook, and a
  conformance reviewer agent. See `plugins/pythonicator/README.md`.
